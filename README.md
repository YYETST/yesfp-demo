### 使用说明

#### 执行方式
直接执行java.demo.test包下的测试类的main方法即可

目前只提供了几个接口测试类
其他接口测试，需要自行在BuildParam.java构建参数，在URLConfigEnum.java下配置url

#### 包含义
- crypto 放置密钥相关的处理类
- entity 放置参数构建类
- test 放置接口测试的入口类
- utils 放置请求相关的工具类
